# Stock Market Analysis & Portfolio Optimization

## 📌 Overview
This project analyzes stock market data to evaluate returns, risk (volatility), and optimize portfolio allocation using financial metrics.

## 🎯 Objective
To identify optimal investment strategies by analyzing risk-return tradeoffs and maximizing Sharpe Ratio.

## 🛠 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- yfinance

## 📊 Key Features
- Stock price trend analysis
- Daily return calculation
- Risk vs Return visualization
- Portfolio return & risk calculation
- Sharpe Ratio evaluation
- Monte Carlo simulation for portfolio optimization

## 💡 Key Insights
- Diversification reduces overall portfolio risk
- Optimal portfolios maximize return per unit of risk
- Risk-return tradeoff is essential in investment decisions
